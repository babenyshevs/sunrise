# Sunrise — Head of Data Science & Analytics

Personal knowledge base for the role. Raw notes go in; structured knowledge, insights, and next steps come out — driven by Claude Code skills.

## Structure

| Path | What lives here |
|---|---|
| `me.md` | Your own profile — the reference point for finding common ground with others. |
| `notes/` | Dated notes — meetings, 1:1s, interviews, ideas. One file per note (`YYYY-MM-DD-slug.md`). |
| `team/` | Profiles for direct reports and the manager. One file per person. |
| `stakeholders/` | Profiles for cross-org people — product owners, senior leaders, peers. |
| `intreview/` | Interview process history (rounds, assignment, HR). |

## Skills

Run with the Skill tool or `/<name>`:

- **capture-note** — paste a raw dump → filed, structured note in `notes/` with decisions, action items, open questions, and people links.
- **people-profile** — create or update a person's profile; appends to a dated interaction log rather than overwriting.

Typical loop: after a meeting, run `capture-note` on your raw notes, then let it hand off to `people-profile` for anyone worth tracking.
